 第一步：超声波测距原理（搞懂它再写代码）

  超声波模块（常见 HC-SR04）有 4 个引脚：

  VCC  → 5V
  GND  → GND
  Trig → 树莓派 GPIO 输出（发信号）
  Echo → 树莓派 GPIO 输入（收回音）

  工作原理：

  你（树莓派）               超声波模块
      │                         │
      ├─ Trig 拉高 10~15μs ───→ │   ← 发出 8 个 40kHz 脉冲
      │                         │
      │←─ Echo 拉高（回音） ─────┤
      │                         │
      │   Echo 高电平的持续时间     │
      │   = 超声波来回飞行的时间      │
      │                         │

  距离公式：
  距离(cm) = Echo高电平时间(秒) × 340(声速m/s) × 100 / 2
                          ↑          ↑             ↑
                      来回时间        声速       除以2(来回变单程)

  ▎ 核心就是：发个脉冲，掐表算Echo高电平持续了多久

  def Distance():
      # 1. 发脉冲: Trig拉低→拉高15μs→拉低
      GPIO.output(TrigPin, GPIO.LOW)
      time.sleep(0.000002)          # 2μs
      GPIO.output(TrigPin, GPIO.HIGH)
      time.sleep(0.000015)          # 15μs
      GPIO.output(TrigPin, GPIO.LOW)

      # 2. 等Echo变高（开始接收回音）
      t3 = time.time()
      while not GPIO.input(EchoPin):
          t4 = time.time()
          if (t4 - t3) > 0.10:      # ⚠️ 超时保护！等100ms还没反应就放弃
              return -1

      # 3. 等Echo变低（回音结束）
      t1 = time.time()
      while GPIO.input(EchoPin):
          t5 = time.time()
          if (t5 - t1) > 0.10:      # ⚠️ 超时保护！高电平超过100ms也放弃
              return -1

      # 4. 计算距离
      t2 = time.time()
      time.sleep(0.01)
      return ((t2 - t1) * 340 / 2) * 100   # 秒×声速÷2×100 = cm

  ⚠️ 注意：这个超时保护是后来加的，All.py 和 FaceColor.py 里的版本就没超时保护，如果 Echo 悬空会卡死。

  Distance_test() — 多次采样取均值

  def Distance_test():
      num = 0
      ultrasonic = []
      while num < 4:              # 测4次
          distance = Distance()
          while int(distance) == -1:          # 超时了就重测
              distance = Distance()
          while int(distance) >= 500 or int(distance) == 0:  # 不合理值也重测
              distance = Distance()
          ultrasonic.append(distance)
          num += 1
          time.sleep(0.01)
      distance = (ultrasonic[1] + ultrasonic[2] + ultrasonic[3]) / 3  # 去掉第0次，取后3次平均
      return distance

  avoid_obstacle() — 避障线程（会开车！）

  def avoid_obstacle():
      while True:
          global obstacle_detected
          distance = Distance_test()
          if distance < 20:           # <20cm 有障碍
              obstacle_detected = True
          else:
              obstacle_detected = False

  这个函数是在线程里一直跑的，主程序通过检查 obstacle_detected 全局变量来决定是否重规划路径。

  📚 超声波模块完整教学

  一、超声波 HC-SR04 工作原理（你考试/答辩用得上）

  发波: Trig 高电平 10~15μs
            ↓
  模块发射 8 个 40kHz 超声波
            ↓
  遇到障碍反射回来
            ↓
  Echo 引脚输出高电平（宽度=来回时间）
            ↓
  距离 = 高电平时间 × 声速(340m/s) / 2

  关键参数：
  - 测距范围：2cm ~ 400cm
  - 精度：约 3mm
  - 声速：340 m/s → 34000 cm/s
  - 最大 Echo 持续时间：约 24ms 对应 400cm
  - 超时保护阈值：如果你的 Echo 高电平超过 100ms 还没结束，说明有问题

  一、超声波 HC-SR04 工作原理（你考试/答辩用得上）

  发波: Trig 高电平 10~15μs
            ↓
  模块发射 8 个 40kHz 超声波
            ↓
  遇到障碍反射回来
            ↓
  Echo 引脚输出高电平（宽度=来回时间）
            ↓
  距离 = 高电平时间 × 声速(340m/s) / 2

  关键参数：
  - 测距范围：2cm ~ 400cm
  - 精度：约 3mm
  - 声速：340 m/s → 34000 cm/s
  - 最大 Echo 持续时间：约 24ms 对应 400cm
  - 超时保护阈值：如果你的 Echo 高电平超过 100ms 还没结束，说明有问题
