- 事实来源：用户确认此前当前小车实测结果；原始终端输出未保留。

- 90° 左转粗转时间：`0.6` 秒。
- 90° 右转粗转时间：`0.5` 秒。
- 180° 左转粗转时间：`1.2` 秒。
- 180° 右转粗转时间：`1.1` 秒。

当前 `GridNavigator` 的 180° 掉头固定使用左旋，因此运行参数采用左转

这些时间只与 `spin_speed = 30` 配套，改变速度、电池状态、轮胎或场地后
必须重新校准，不能单独复制时间值。
