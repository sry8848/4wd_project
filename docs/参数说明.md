速度参数都是 PWM 占空比，范围 `0～100`。数值越大，电机输出通常越强，但不等于实际速度百分比。

### 路线参数

| 参数          | 当前值    | 说明                          |
| ----------- | ------ | --------------------------- |
| `--rows`    | `5`    | 网格行数，对应 `A～E`               |
| `--cols`    | `5`    | 网格列数，对应 `1～5`               |
| `--start`   | `A1`   | 小车当前所在的可信起点                 |
| `--end`     | `A3`   | 本次导航终点                      |
| `--heading` | `east` | 小车初始车头朝向；`east` 表示从 A1 朝 A2 |

### 巡线参数

| 参数                        | 当前值     | 说明                          |
| ------------------------- | ------- | --------------------------- |
| `--forward-speed`         | `5`     | 走边直行和驶出节点时的基础速度             |
| `--line-turn-speed`       | `80`    | 通用巡线修正速度；指定左右独立速度后主要作为默认回退值 |
| `--line-left-turn-speed`  | `80`    | 左侧检测到黑线时的左修正速度              |
| `--line-right-turn-speed` | `100`   | 右侧检测到黑线时的右修正速度，用于补偿左右动力差异   |
| `--search-speed`          | `5`     | 转向后精细寻找新边黑线的原地旋转速度          |
| `--delay`                 | `0.02s` | 巡线状态机每轮读取传感器的间隔             |

### 转向参数

| 参数                           | 当前值    | 说明                   |
| ---------------------------- | ------ | -------------------- |
| `--spin-speed`               | `30`   | 左转、右转和掉头预转阶段的原地旋转速度  |
| `--left-turn-rough-seconds`  | `0.4s` | 左转时先按计划方向粗略预转的时间     |
| `--right-turn-rough-seconds` | `0.3s` | 右转时先粗略预转的时间          |
| `--uturn-rough-seconds`      | `0.8s` | 180°掉头时的左旋预转时间       |
| `--turn-acquire-timeout`     | `5.0s` | 预转后精细找线的最长时间；超时则停车失败 |

预转时间故意小于完整转弯时间。预转完成后，程序继续根据巡线传感器寻找新方向的黑线，而不是完全依赖固定时间转到目标角度。

### 出点和入点

| 参数                         | 当前值     | 说明                           |
| -------------------------- | ------- | ---------------------------- |
| `--leave-node-min-seconds` | `0.10s` | 离开旧交叉点前至少运动的时间，防止把旧节点立即当成新节点 |
| `--node-clear-samples`     | `1`     | 读取一次符合离开节点的全白状态，即确认已经出点      |
| `--node-confirm-samples`   | `1`     | 读取一次节点状态，即确认到达交叉点            |
| `--node-center-speed`      | `20`    | 识别节点后，朝车头方向前推居中的速度           |
| `--node-center-seconds`    | `0.10s` | 节点居中前推持续时间                   |
| `--line-acquire-timeout`   | `3.0s`  | 出点状态允许的最长时间，超时则停车            |
| `--line-lost-timeout`      | `5.0s`  | 走边阶段持续全白的安全上限，超过后按丢线停车       |

节点到达后的流程是：

```
识别交叉点
→ 刹车
→ forward(20,20) 前推0.10秒
→ 刹车
→ 更新节点并执行下一次转向
```

正常前进入点和倒车返回节点都使用这套居中动作。

### 障碍与倒车

| 参数                           | 当前值    | 说明                      |
| ---------------------------- | ------ | ----------------------- |
| `--threshold`                | `20cm` | 有效距离小于 20cm 时，本次读数判为障碍  |
| `--obstacle-confirm-samples` | `2`    | 连续两次不同序号的测距都发现障碍，才封锁当前边 |
| `--reverse-speed`            | `5`    | 遇障碍后沿原边直线倒车的速度          |
| `--reverse-turn-speed`       | `20`   | 倒车巡线时左右偏移修正速度           |
| `--recovery-timeout`         | `8s`   | 倒车返回原节点的最长时间，超时必须停车     |

安全距离读数会把障碍累计次数清零。`-1` 仍表示无有效距离，不直接判断为障碍。

### 整体超时与调试

| 参数               | 当前值   | 说明                     |
| ---------------- | ----- | ---------------------- |
| `--edge-timeout` | `20s` | 单条计划边的总期限，包括转向、出点和走边   |
| `--debug`        | 开启    | 输出路径规划、转向、节点居中、障碍和恢复状态 |
| `--line-debug`   | 开启    | 每轮输出四路巡线读数、判断动作和电机命令   |

当前命令没有添加以下开关，因此对应功能保持开启：

| 未添加的开关               | 当前效果     |
| -------------------- | -------- |
| `--no-ultrasonic`    | 超声障碍检测开启 |
| `--no-reverse-radar` | 倒车蜂鸣雷达开启 |
